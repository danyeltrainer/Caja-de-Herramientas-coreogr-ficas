# Caja-de-Herramientas-coreogr-ficas
Material de apoyo para el uso y desarrollo de la composición coreográfica y de rutinas
Este proyecto es una infografía HTML sobre herramientas coreográficas, que incluye:

Tema 1: Effort-Shape (Laban) - Analiza el movimiento en cuatro factores: Peso, Tiempo, Espacio y Flujo.

Tema 2: Polimocionalidad - Explora la superposición de emociones y movimientos.

Tema 3: Estructuras - Presenta formas de organizar el movimiento (ABC, Rondó, Canon, etc.).

Tema 4: Lenguaje - Describe los bloques de construcción del movimiento (Motivo, Frase, Secuencia).

El archivo es autónomo: tiene estilos integrados (Tailwind CSS, Font Awesome) y está listo para abrir en un navegador. También tiene un botón para imprimir las láminas.
